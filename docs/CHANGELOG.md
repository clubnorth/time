# 时间轴 · 变更日志

---

## [1.0.0] — 2026-07-11 — 正式发布

### 功能
- **时间轴视图**：按日期分组，逆序展示记录
- **13 种条目类型**：
  - 随记（太阳/阴雨，SVG 图标）、运动（跑步/骑行/俯卧撑/跳绳/游泳/徒步/自由活动）、自律打卡、禁糖打卡、尿酸记录、资产记录、读书（在读/已读）、影视（电影/剧集/动漫）、完成待办
- **智能数据增强（DeepSeek AI）**：
  - 已读书籍自动查询作者、国籍、字数、首次出版年份、标签
  - 影视作品自动查询导演、主演、国家、上映年份、时长/集数、标签
- **评分系统**：1-10 分，星星自动转换（2分=1星），半星支持
- **图书分类**：推理悬疑、明日方舟、人物历史、社会科学、小说文学、自我成长、杂项拾遗
- **待办事项**：
  - 截止时间全自动分类（你忘记做了 / 今日 / 未来三日 / 以后）
  - 整点/半点时间选择
  - 左滑修改/删除，完成联动写入时间轴
- **补卡回填**：选择历史日期补录打卡，自动重算连续天数
- **统计分析**：8 类 GitHub 风格热力图 + 统计卡片
- **面板排序**：添加面板条目自由排序，持久化到数据库
- **时间轴长按删除**：长按 3 秒弹出气泡确认删除

### 技术
- Vue 3 + Vite 前端，Go + SQLite (modernc.org/sqlite) 后端
- RESTful JSON API + DeepSeek API 集成
- 数据库版本化迁移系统（3 个迁移）
- 数据导出/导入 API
- CSS 变量响应式设计（手机/平板/桌面四断点）
- Ink on Paper 设计主题

### 数据模型
- `entries` — 13 种类型的时间轴记录
- `todos` — 含 due_date / category 的待办事项
- `settings` — 键值对配置
- `schema_migrations` — 迁移版本追踪

### 性能
- IndexedDB-free，纯 SQLite 单文件数据库
- CGo-free 编译，可交叉编译到任意平台
- 前端 118 kB JS / 35 kB CSS（gzip 后 40 kB / 5.8 kB）

### 已知限制
- 单用户，无身份验证
- DeepSeek API 密钥明文存于 config.json
